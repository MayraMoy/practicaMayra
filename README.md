## Descripción

Proyecto práctico realizado para la materia de Programación III utilizando Java, Spring Boot, JPA e Hibernate.

>[!NOTE]
>El objetivo es comprender la construcción de APIs REST, persistencia de datos y relaciones entre entidades utilizando una arquitectura backend moderna.


# Temas trabajados

**Spring Boot & Configuración**

- Spring Initializr
- Maven y gestión de dependencias
- Estructura de proyectos Spring Boot
- `application.properties`
- Variables de entorno
- Configuración de puertos
- DevTools

## Base de Datos

- MySQL
- MySQL Workbench
- JDBC
- Persistencia de datos
- Hibernate DDL Auto (`update`)

## JPA & Hibernate

- ORM
- JPA
- Hibernate
- Entidades (`@Entity`)
- Tablas (`@Table`)
- Claves primarias (`@Id`)
- Generación automática (`@GeneratedValue`)
- Columnas (`@Column`)
- Relaciones entre entidades:
  - `@OneToOne`
  - `@OneToMany`
  - `@ManyToOne`
  - `@ManyToMany`
- Claves foráneas (`@JoinColumn`)
- Cascade
- `orphanRemoval`
